# 생활금융 계산소

대출 이자, 부가가치세, 연봉 실수령액, 퇴직금 계산기를 하나의 정적 웹사이트로 통합한 초안입니다.

## 포함 페이지

- `index.html`: 홈
- `loan.html`: 대출 이자 계산기
- `vat.html`: 부가세 계산기
- `salary.html`: 연봉 실수령액 계산기
- `severance.html`: 퇴직금 계산기
- `guide.html`: 계산 기준과 출처 안내
- `about.html`, `privacy.html`, `contact.html`: 심사 및 운영 기본 페이지

## 공개 전 체크리스트

- 브랜드명과 사용할 도메인을 최종 확정하고 메타 정보 및 사이트맵을 추가합니다.
- `contact.html`에 실제 수신 가능한 이메일 또는 문의 폼을 연결합니다.
- 배포 후 모든 페이지 링크, 모바일 표시, 계산 결과를 다시 검증합니다.
- AdSense 승인을 신청하기 전 개인정보처리방침에 실제 광고/분석 도구 사용 내용을 일치시킵니다.
- 공식 보험요율과 세무 기준이 변경되면 `assets/app.js`와 `guide.html`을 함께 업데이트합니다.

## 계산 기준

- 연봉 계산기의 국민연금 근로자 부담률은 2026년 사업장가입자 기준 `4.75%`를 적용합니다.
- 건강보험 및 장기요양보험은 2026년 공개 요율을 기준으로 추정합니다.
- 퇴직금은 세전 금액만 계산하며 퇴직소득세는 포함하지 않습니다.
- 모든 계산 결과는 참고용이며 신고, 계약 또는 급여 정산의 공식 결과를 대체하지 않습니다.
